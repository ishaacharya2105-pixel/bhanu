# **App Name**: LexiLearn

## Core Features:

- Word Entry: Users can add new words along with their definitions and example sentences.
- Flashcard Practice: Practice vocabulary using interactive flashcards.
- Vocabulary Quizzes: Test word recall through short, engaging quizzes.
- Progress Tracking: Monitor learning progress over time with visual charts and statistics.
- Contextual Usage Tool: Use a tool powered by generative AI to show relevant usages of each vocab word based on general context and/or user input

## Style Guidelines:

- Primary color: Light blue (#64B5F6) to evoke a sense of calm and focus.
- Background color: Very light blue (#E3F2FD) to keep the interface airy and non-distracting.
- Accent color: Slightly darker blue (#42A5F5) for interactive elements and highlights.
- Body and headline font: 'PT Sans' (sans-serif) for a clean and modern readability.
- Use clean, minimalist icons to represent different functions and categories.
- Ensure a clean, spacious layout to make it easy to focus on learning without distractions.
- Incorporate subtle transitions and animations to enhance user engagement without being overwhelming.
